# 展厅系统-功能清单

## 功能清单

| 大模块 | 小模块 | 功能描述 |
| --- | --- | --- |
| 展厅使用申请 | 新建申请 | 录入会议主题、简要议程、使用起止时段、来访单位、所属行业、人员数量、最高陪同领导、申请人、申请部门、联系电话、备注 |
|  | 区县与实名标识 | 因多区县共用账号,必填所属区县、真实姓名、手机号,操作全程留痕 |
|  | 时段冲突检测 | 提交前自动检查展厅时段是否已被占用,冲突则阻止提交并提示占用详情 |
|  | 附加服务勾选 | 桌牌打印、茶水、合影、拍照、车辆接送等附加服务选项 |
|  | 多客户登记 | 支持单次申请填写多家来访单位,对齐数据表多客户多行 |
| 审批流转 | 动态审批链 | 一级部门领导 → 二级展厅主管 → 三级按陪同领导级别动态追加(分管副总/总经理) |
|  | 节点动作 | 同意、驳回(带意见)、转交 |
|  | 待办提醒 | 站内消息提醒审批人待办,提交人与展厅主管同步通知 |
|  | 前一天提醒 | 参观前一天提醒展厅主管准备 |
|  | 超时审批升级 | 节点停留超阈值自动催办或升级 |
| 排期日历 | 日历视图 | 日/周/月视图显示已审批、待审批、已完成、已取消 |
|  | 拖拽改期 | 日历内拖拽触发改期审批子流程 |
|  | 并行场次 | 展厅容量与并行场次校验 |
| 异常处理 | 取消参观 | 申请人发起取消并填写原因,审批中或已通过均可,释放排期 |
|  | 改期 | 简化审批子流,保留原单号并记录变更日志 |
|  | 临时到访补录 | 无申请的客户到访支持事后补录申请与补录参观记录 |
|  | 超期未归还催办 | 使用结束未归还自动催办,展厅主管可代为归还登记 |
|  | 陪同领导变更追加审批 | 领导级别上调(如分管副总→总经理)触发追加审批 |
|  | 实际人数记录 | 实际到场人数 vs 计划人数对比留痕 |
|  | 同行业客户避让 | 同日同行业客户参观时提醒,避免竞品同场 |
|  | 不可抗力批量处理 | 管理员按时段批量改期或取消(台风/疫情等) |
| 参观留存 | 留存录入 | 实际参观时间、实际参观人数、我方领导参会最高级别 |
|  | 客户与战客信息 | 参观客户单位、全网客户编码、所属行业、是否战客、战客级别(省/市/区县直管/其他) |
|  | 商机编码录入 | 触发商机编码,允许填'无'并后续补录 |
|  | 现场照片上传 | 多图上传至 MinIO,替代 Excel DISPIMG,H5 支持直接拍摄 |
|  | 归还签收 | 展厅主管录入归还签字人与归还时间 |
| 商机跟进 | 商机编码补录 | 参观记录后续追加或修改商机编码 |
|  | 商机状态跟踪 | 线索 → 商机 → 签约 状态流转 |
|  | 一次参观多商机 | 支持一次参观关联多条商机 |
| 报表与看板 | 战客覆盖率看板 | 战客参观家数 / 省市直管战客数,年度覆盖率(≥50% 目标) |
|  | 分布统计 | 按月、行业、战客级别、地市分布 |
|  | 商机转化漏斗 | 参观 → 商机 → 签约 转化漏斗图 |
|  | 督办清单 | 超期未录商机编码清单 |
|  | 省公司报送导出 | 一键导出 Excel,列结构严格对齐原 xlsx Sheet2 |
| 基础数据 | 字典维护 | 省代码、地市编码、行业类型、战客级别(对齐枚举值 Sheet) |
|  | 客户档案 | 客户信息与全网客户编码映射 |
|  | 部门人员 | 部门、人员、角色权限(基于 Pig RBAC) |
|  | 展厅参数 | 单展厅基础信息通过系统参数配置(名称、地址、负责人) |
| 附件与文档 | docx 回填生成 | 按原模板生成存档版申请单 docx(poi-tl) |
|  | 附件管理 | 议程 PDF、会议资料等附件上传 |
|  | 审批意见留痕 | 全链路审批意见与时间留存 |
